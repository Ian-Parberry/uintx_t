Overview

A program to compute arithmetic expressions using uintx_t. Enter a parenthesized arithmetic expression using decimal unsigned integers of any length, the arithmetic operators \(+-*/\) with normal operator precedence, and the functions listed in the table below:

Function Result
\(\text{sqrtx}(n)\)
\(\lfloor \sqrt{n} \rfloor\)
\(\text{sqrx(n)}\)
\(n^2\)
\(\text{factorialx(n)}\)
\(n!\)
\(\text{exp2x(n)}\)
\(2^n\)
\(\text{fibx(n)}\)
\(n^\text{th}\) Fibonacci number
\(\text{log2x(n)}\)
\(\lfloor \log_2 n \rfloor\)

Settings can be adjusted by typing in these commands:

Command Setting
verbose Verbose mode.
quiet Quiet mode.
decimal Print results in decimal.
hex Print results in hexadecimal.
commas Print results in decimal with commas.
quit Exit.
