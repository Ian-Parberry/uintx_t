Detailed Description

Each node can represent an operator or a number. If it is an operator, m_bIsOperator is true, m_chOp is a character representing an operator, and m_pLeftChild and m_pRightChild point to the operands. If it is a number, m_bIsOperator is false and m_nValue contains the value.
