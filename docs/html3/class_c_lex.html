Detailed Description

A simple lexical analyzer for arithmetic expressions. A lexical analyzer breaks up the character input into tokens. Each token consists of a SymbolType accompanied by a uintx_t if the symbol type is SymbolType::Number, and a string containing an identifier name if the symbol type is SymbolType::Identifier.
