◆ fibx()

const uintx_t fibx ( uint32_t  n)

Fibonacci numbers computed using successive doubling, that is, using the recurrences \(F(0) = F(1) = 0\) and for all \(k \geq 2\),

\[F(2k) = F(k)(2F(k+1)-F(k))\]

\[F(2k+1) = F(k+1)^2 + F(k)^2\]

.

Parameters

Returns
